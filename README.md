## 🚀 快速开始
### 1. 开发环境要求
- **微信开发者工具**：用于小程序开发、调试和预览
- **下载地址**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 运行步骤
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择目录：`e:\k\miniprogram`
4. 填写项目名称（例如：AI色彩构图学习工坊）
5. AppID选择"测试号"（无需注册）
6. 点击"确定"导入
7. 等待项目编译完成
8. 在模拟器中即可预览
9. 点击"预览"按钮生成二维码，手机微信扫码体验

### 3. 演示数据说明
首次运行时，应用会自动初始化演示数据：
- **3个示例作品**：包含完整演示不同风格的作品
- **5个学习任务**：从易到难的任务列表
- **9个知识卡片**：涵盖色彩理论、构图法则、设计原则
- **用户账号**：默认用户"学习达人"
