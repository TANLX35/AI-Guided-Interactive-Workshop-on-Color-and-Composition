# 图片资源说明

本目录需要包含以下图标文件,用于小程序底部导航栏(TabBar):

## 需要添加的图标文件

所有图标建议尺寸:81×81像素

### 首页图标
- `home.png` - 首页默认图标
- `home-active.png` - 首页激活状态图标

### 沙盒图标
- `sandbox.png` - 沙盒默认图标
- `sandbox-active.png` - 沙盒激活状态图标

### 任务图标
- `tasks.png` - 任务默认图标
- `tasks-active.png` - 任务激活状态图标

### 理论图标
- `knowledge.png` - 理论默认图标
- `knowledge-active.png` - 理论激活状态图标

### 个人中心图标
- `profile.png` - 个人中心默认图标
- `profile-active.png` - 个人中心激活状态图标

## 图标设计说明（文艺风格）

1. **风格**: 简约线条风，透明背景，适合文艺清新配色体系
2. **颜色**:
   - 默认图标: 暖棕色 #B5A898（与 tabBar color 一致）
   - 激活图标: 深棕色 #8B7B6B（与 tabBar selectedColor 一致），带半透明填充
3. **内容**:
   - 首页: 房子轮廓图标
   - 创作: 画板+画笔图标
   - 任务: 清单/复选框图标
   - 理论: 打开的书本图标
   - 我的: 人物头像轮廓图标

## 如何添加图标

1. 使用设计工具(如Figma、Sketch)创建图标
2. 导出为PNG格式,尺寸81×81像素
3. 将文件放置在本目录中
4. 确保文件名与上述列表一致

## 临时解决方案

如果暂时没有图标文件,可以:
1. 使用纯色方块作为临时图标
2. 使用文字代替图标(修改app.json中的tabBar配置)
3. 从图标库网站下载免费图标(如iconfont.cn)

## 注意事项

- 图标文件必须存在,否则小程序无法正常运行
- 建议使用透明背景的PNG文件
- 激活状态图标应与默认图标有明显区别
